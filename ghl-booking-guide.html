GHL Booking Page Guide (Simple English)

This guide helps you recreate booking.html inside GoHighLevel (GHL) and collect payments.

1) Payment Setup

  1. Go to Settings > Integrations.
  2. Connect Stripe (required for online payments).

2) Products & Prices

Create products for each class. Add two prices:

  • Monthly
  • Upfront (one-time)
Tip: Use the class name as the product name. Example: "Hip Hop Upskill - Wednesday 6:30 PM".

3) Order Form (One-Step)

Create a one-step Order Form so the user fills the form and pays on the same page.

Add payment to the form

  1. Go to Payments > Order Forms and open your form.
  2. In the form builder, add or select the Order element (this is the payment section).
  3. Under Products / Offers, add your class products with two prices: Monthly and Upfront.
  4. Set the Order Form to One-Step so customer details + payment happen together.
  5. Enable Stripe for the payment method.
  6. Save the form.

Form fields (from booking.html)

  • Student First Name
  • Student Last Name
  • Date of Birth / Age
  • Parent/Guardian Name
  • Parent/Guardian Phone
  • Parent/Guardian Email
  • Medical / Learning Needs (optional)
  • Emergency Contact (optional)
  • Checkbox: I agree to policies and consent form

4) Booking Page (Website)

  1. Go to Sites > Websites.
  2. Create a page named booking.
  3. Drag your Order Form onto the page.
  4. Use the same headings and labels as in booking.html.

5) Success Message (Same Page)

On the Order Form settings, show a success message on the same page.

Example success message:
"Thank you! Your booking is confirmed. A confirmation email and WhatsApp message have been sent."

6) Email + WhatsApp Automation

  1. Go to Automation > Workflows.
  2. Trigger: Order Submitted or Payment Successful.
  3. Actions:
    • Send Email (confirmation)
    • Send WhatsApp message
    • Tag contact (e.g. booked-class)
