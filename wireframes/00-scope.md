MCR Dance & Cheer Academy - Wireframe Scope (MVP)

Goal
- Create an internal booking flow replacing Coacha.
- No parent accounts; capture student details only.
- Unlimited class capacity.
- Stripe payments (card + Apple Pay + Google Pay via Stripe).
- Pricing: term/course price with monthly installments and full upfront (10% discount).

Primary User Flows
A) Book a class (paid)
B) Book a free trial
C) Manage booking via email link (change/cancel)

Core Pages (MVP)
1) Home
2) Classes (list + filters)
3) Class Detail
4) Booking + Payment
5) Booking Confirmation
6) Manage Booking (email + OTP)

Supporting Pages/Sections (MVP)
- About (mission, team, coaching quality)
- Contact (location, map, phone, email, form)
- Policies / FAQ (cancellation, make-up, payment terms)

Shared Components
- Header: logo, nav, CTA buttons (Book a class, Free trial)
- Footer: contact, socials, address, policies, newsletter
- Class card: style, age, day/time, price, CTA
- Banner: free trial availability
- Trust: testimonials, coaching qualifications, community/awards
